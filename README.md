# Library Management System
## Description

The Library Management System is a C++ application using SQLite, designed to automate and simplify library operations. It helps manage, search, and process information about books and users, improving efficiency for librarians and convenience for visitors.

## Purpose

- Organize and manage library resources.

- Automate daily library tasks.

- Provide a modern and user-friendly interface.

## Key Features

- User authentication with access control (admin, user).

- Viewing and managing the book catalog.

- Searching books by various parameters.

- Automated book issuing and returning.

- Operation logging and inventory updates.

- Graphical interface for easy navigation.

## Technologies

- C++

- SQLite

- Qt framework
